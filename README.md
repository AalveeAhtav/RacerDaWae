How to Compile/Run
    Requirements
        Unity Hub + Unity Editor version 6000.3.11f1
        OR Windows or Mac OS (for .exe build)

    Option A - Run the executable
        Download and unzip the submission
        Open the Game folder
        Double click the InitialDriftRacing.exe to directly launch the game

    Option B - Run from source in Unity
        Clone or download the repository from GitHub: https://github.com/AalveeAhtav/RacerDaWae
        Open Unity Hub and click Add → Add project from disk
        Go to the downloaded SourceCode folder and select 
        Once open, double click MainScene in Assets/Scenes to open it
        Press the Play button at the top of the Unity Editor to run the game

    Controls
        W : Accelerate
        A/D : Steer Left/Right
        S : Brake/Reverse
        Space: Handbrake
        ESC : Pause menu
